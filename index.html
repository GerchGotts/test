// Кодекс

Как здесь устроено

  1. 01

    Отыгрывай свой сюжет, не скачивая дополнительных модов для сервера.

  2. 02

    Плагины и новые механики помогут в реализации большинства идей.

  3. 03

    Администрация не щадит игроков. Сложность и напряжение — часть опыта, а не недостаток.

  4. 04

    Инициатива ценится выше выполнения инструкций.

  5. 05

    Игроки сервера — команда. Честность важнее показной дружелюбности.

  6. 06

    Игроки всегда помогут в освоении сюжета.

// История

Основной сюжет мира

Начало истории

Два соседних государства не могут поделить территорию, находящуюся между ними. Именно поэтому вас — маргиналов, которых было не жалко туда отправить, — сослали исследовать эти земли.

Теперь вам предстоит выживать в неизвестном мире, встречать путников, выбирать союзников и врагов и самостоятельно писать историю GAP.

// Моды

Что нужно для сервера

  • 01

    Вам не обязательно скачивать ни один из приведенных примеров модов.

  • 02

    Custom Player Model (CPM). Нужен для отображения эксклюзивных скинов игроков.

  • 03

    Emotecraft. Для возможности выразить свои эмоции буквально.

  • 04

    Dynamic Lights. Чтоб вашу дорогу освещала не только надежда.

  • 05

    Inventory Profiles Next. Автототем не запрещён на сервере.

  • 06

    Flashback. Запечатлеть самые забавные моменты чужих фейлов.
